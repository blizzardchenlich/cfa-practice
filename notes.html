Quantitative Methods · Reading 1
1. Rates and Returns
Interest Rates
HPR & Mean Returns
Money- vs Time-Weighted
Interest rate = required return = discount rate = opportunity cost
  • Nominal rate ≈ real risk-free rate + inflation premium
  • Required return = real risk-free + inflation + default + liquidity + maturity risk premiums
  • Real interest rates are inflation-adjusted
Holding Period Return:
HPR = (P₁ − P₀ + D) / P₀
  • Arithmetic mean: simple average; best for single-period estimates
  • Geometric mean: compound annual rate; multi-period performance
  • Harmonic mean X̄_H = n / Σ(1/xᵢ); used for average cost of equal periodic investments
  • Trimmed: remove top/bottom extreme %; Winsorized: replace extremes with threshold values
Money-Weighted Return (MWR):
  • = IRR of all cash flows into/out of portfolio
  • Sensitive to timing of cash flows
  • Use when manager controls cash flows
Time-Weighted Return (TWR):
  • = geometric mean of sub-period HPRs
  • Not affected by cash flow timing
  • Preferred measure for evaluating manager performance
Annualized & Compounded Returns
Common Return Measures
Annualized Return:
(1 + HPR)^(365/days) − 1
Continuously Compounded Return (R_cc):
R_cc = ln(1 + HPR) = ln(ending value / beginning value)
  • Additive across time periods
  • As compounding frequency ↑ → FV ↑ and PV ↓
  • Gross return: total return before management fees (after trading costs)
  • Net return: after all fees and expenses
  • Pretax nominal return: before taxes and inflation adjustment
  • After-tax nominal return: after tax liability, before inflation
  • Real return: inflation-adjusted; measures purchasing power gain
  • Leveraged return = gain or loss / investor's own cash invested

2. The Time Value of Money in Finance
PV / FV Formulas
Fixed-Income Securities
Equity Valuation (DDM)
FV = PV × (1 + r)ᵗ
PV = FV / (1 + r)ᵗ = FV × (1 + r)⁻ᵗ
Perpetuity:
PV = PMT / r
Annuity payment:
PMT = r × PV / [1 − (1+r)⁻ᵗ]
Continuous compounding:
FV = PV × eʳᵗ    PV = FV × e⁻ʳᵗ
  • Zero-coupon bond: PV = FV/(1+r)ᴺ  (no periodic payments)
  • Coupon bond: PV = Σ [PMT/(1+r)ᵗ] + FV/(1+r)ᴺ
  • Price ↑ ↔ Yield ↓ (inverse relationship)
  • Perpetual bond: PV = PMT/r
  • Amortizing bond: each payment includes principal portion (annuity)
  • Negative yield → bond priced at premium (price > face value)
Preferred stock (perpetuity):
V = D_p / k_p
Gordon Growth Model (constant growth):
V₀ = D₁ / (k_e − g_c)
  • Implied return: k_e = D₁/V₀ + g_c
  • Implied growth: g_c = k_e − D₁/V₀
  • Multistage DDM: discount high-growth dividends individually, then apply constant-growth model for terminal value at transition point
Cash Flow Additivity
Forward Interest Rates
Forward Exchange & Option Pricing
  • PV of a stream = sum of PVs of each individual cash flow
  • Can split or combine cash flow series; PVs still add up
  • Basis for the no-arbitrage principle ("law of one price")
  • Two identical future cash flow streams must have the same price today
  • Spot rate S₁: for loan made today
  • Forward rate notation: 1y1y = 1-year loan 1 year from now
  • (1 + S₂)² = (1 + S₁)(1 + 1y1y)
  • 1y1y = (1+S₂)²/(1+S₁) − 1
  • Forward rates are implied by observable spot rates
No-arbitrage forward exchange rate:
F/S = (1 + r_price) / (1 + r_base)
Binomial option pricing:
  • Form hedge portfolio: write 1 call, buy h shares
  • h (hedge ratio) = (c_u − c_d) / (S_u − S_d)
  • No-arbitrage gives call value c₀
  • Key insight: equivalent payoffs must have same present value

3. Statistical Measures of Asset Returns
Central Tendency
Location & Quantiles
Dispersion Measures
  • Arithmetic mean: X̄ = ΣXᵢ/n; most widely used
  • Median: middle value when sorted; robust to outliers
  • Mode: most frequent value; can be bimodal/trimodal
  • Trimmed mean: exclude top & bottom percentage
  • Winsorized mean: replace extremes with threshold values (n−1 denominator still used)
  • Quantile: value below which x% of data falls
  • Quartile: Q1 = 25th, Q2 = 50th, Q3 = 75th percentile
  • Quintile: 5 equal groups; Decile: 10 equal groups
  • IQR = Q3 − Q1 (interquartile range)
  • Box and whisker plot: visualizes IQR, median, range
  • Range = max − min
  • MAD = Σ|Xᵢ − X̄| / n
  • Sample variance:
    s² = Σ(Xᵢ−X̄)² / (n−1)
  • Std deviation s = √s² (same units as data)
  • CV = s/X̄ (risk per unit return; lower = better)
  • Target semideviation:
    s_target = √[Σ(Xᵢ−B)²/(n−1)] for Xᵢ<B
Skewness
Kurtosis
Correlation & Covariance
  • Positive (right) skew: Mean > Median > Mode; long right tail; large positive outliers pull mean up
  • Negative (left) skew: Mean < Median < Mode; long left tail
  • Symmetric: Mean = Median = Mode
  • |Skewness| > 0.5 → significant skew
  • Skewness = 0 for normal distribution
  • Leptokurtic: more peaked, fat tails, excess kurtosis > 0 → greater extreme-outcome risk
  • Platykurtic: less peaked, thin tails, excess kurtosis < 0
  • Mesokurtic: normal distribution (kurtosis = 3; excess kurtosis = 0)
  • Securities returns tend to be leptokurtic → models underestimate tail risk
Covariance:
Cov(X,Y) = Σ{(Xᵢ−X̄)(Yᵢ−Ȳ)} / (n−1)
Correlation coefficient:
ρ_XY = Cov(X,Y) / (s_X × s_Y)   ∈ [−1, +1]
  • ρ = 0 → no linear relationship
  • Correlation ≠ causation
  • Spurious correlation: due to chance or a third variable
  • Scatter plots reveal non-linear relationships

4. Probability Trees and Conditional Expectations
Expected Value & Variance
Probability Trees
Bayes' Formula
Expected value (probability model):
E(X) = ΣP(xᵢ) × xᵢ
Variance:
Var(X) = ΣP(xᵢ) × [xᵢ − E(X)]²
  • σ = √Var(X)
  • Forward-looking model (uses probabilities, not historical n−1)
  • Visualizes sequential conditional events with branches
  • Joint probability = product of branch probabilities along path
  • All terminal probabilities sum to 1
  • Conditional expected value: E(X|event) — revises expectations given new information
  • Example: economy state → company results → EPS scenarios
Bayes' Formula (updating probabilities):
P(A|B) = P(B|A) × P(A) / P(B)
  • Updates prior probability P(A) given new event B
  • P(B) = unconditional probability of new information
  • P(A|B) = posterior (updated) probability
  • Used to revise investment probabilities as new data arrives

5. Portfolio Mathematics
Portfolio Return
Portfolio Variance
Safety-First Criterion
Expected portfolio return:
E(Rp) = Σwᵢ × E(Rᵢ)
  • wᵢ = weight of asset i; Σwᵢ = 1
  • Weighted average of individual expected returns
2-Asset Portfolio variance:
σ²p = w₁²σ₁² + w₂²σ₂² + 2w₁w₂Cov(R₁,R₂)
Cov(R₁,R₂) = ρ₁₂σ₁σ₂
  • Diversification benefit: ρ < 1 reduces portfolio variance
  • ρ = −1: maximum diversification benefit
  • ρ = +1: no diversification benefit
Roy's Safety-First Ratio:
SFR = [E(Rp) − R_L] / σp
  • R_L = threshold (minimum acceptable) return
  • Choose portfolio with highest SFR
  • Minimizes P(Rp < R_L)
  • If returns ~ normal: P(Rp < R_L) = N(−SFR)
  • Equivalent to Sharpe ratio when R_L = risk-free rate

6. Simulation Methods
Lognormal Distribution
Monte Carlo Simulation
Bootstrap Resampling
  • If ln(X) ~ Normal, then X is lognormally distributed
  • Asset prices are modeled as lognormal (prices ≥ 0)
  • Continuously compounded returns → normal distribution → asset prices → lognormal
  • Right-skewed; bounded below by zero
  • E.g., stock price after 1 year = S₀ × e^(μ+σZ)
  • Generate many random scenarios using specified probability distributions
  • Estimate distribution of possible outcomes (e.g., portfolio value, option price)
  • Uses: option pricing, VaR, pension fund modeling
  • Advantage: handles complex, path-dependent payoffs
  • Limitation: results only as good as model assumptions; garbage-in → garbage-out
  • Bootstrap: repeatedly sample with replacement from observed data; estimate sampling distribution empirically
  • Jackknife: leave-one-out approach; estimate parameter by omitting one observation at a time
  • Both are non-parametric — no distribution assumption needed
  • Useful when parametric assumptions (normality) may not hold

7. Estimation and Inference
Sampling Methods
Central Limit Theorem
Resampling Techniques
  • Simple random: each element equally likely; minimal bias
  • Stratified random: divide into sub-groups (strata), sample proportionally from each
  • Cluster: randomly select entire clusters; cost-effective but less precise
  • Convenience: use available data; subject to selection bias
  • Judgmental: expert selects; subjective bias risk
Central Limit Theorem (CLT):
  • For large n: sample mean X̄ is approximately normally distributed regardless of the shape of original distribution
  • Mean of sampling distribution = population mean μ
  • Standard error: σ_X̄ = σ/√n
  • Larger n → smaller standard error → more precise estimate
  • Foundation for confidence intervals and hypothesis testing
  • Bootstrap: sample with replacement; build empirical sampling distribution; useful when distribution unknown
  • Jackknife: omit one observation at a time; recompute statistic; estimate bias and variance
  • Both overcome need for parametric distributional assumptions
  • Bootstrap preferred for complex statistics; jackknife better for variance estimation
Quantitative Methods · Reading 8
8. Hypothesis Testing
Setup & Decision Rules
Errors & Power
Test Statistics
  • H₀ (null): hypothesis to be tested (e.g., μ = 0); researcher seeks to reject it
  • H_a (alternative): accepted if H₀ rejected
  • One-tailed: tests direction (H_a: μ > 0 or μ < 0)
  • Two-tailed: tests any difference (H_a: μ ≠ 0)
  • Significance level α: maximum acceptable P(Type I error), typically 5% or 1%
  • Reject H₀ if |test statistic| > critical value, or p-value < α
  • Type I error: reject true H₀ → probability = α (significance level)
  • Type II error: fail to reject false H₀ → probability = β
  • Power of test = 1 − β = P(correctly rejecting false H₀)
  • Tradeoff: ↓α → ↑β (lower Type I error raises Type II error)
  • p-value: smallest α at which H₀ is rejected; reject H₀ if p-value < α
  • z-test: σ² known, or n ≥ 30
  • t-test: σ² unknown, small sample; df = n−1
  • Chi-square (χ²) test: test variance; goodness of fit; independence; df = n−1
  • F-test: equality of two variances; df = (n₁−1, n₂−1)
  • Parametric tests: assume specific distributions (more powerful if assumptions hold)
  • Non-parametric tests: fewer assumptions; use for ordinal or non-normal data

9. Parametric and Non-Parametric Tests of Independence
Parametric: Correlation Test
Non-Parametric: Chi-Square Test
Parametric vs Non-Parametric
Test H₀: ρ = 0 (no linear correlation)
t = r√(n−2) / √(1−r²)   df = n−2
  • Reject H₀ if |t| > critical t-value
  • Also: Spearman rank correlation (non-parametric alternative for ranked data)
χ² Test of Independence (contingency table):
χ² = Σ [(O − E)² / E]
  • O = observed, E = expected frequency
  • df = (rows − 1)(cols − 1)
  • Reject H₀ (independence) if χ² > critical value
  • All expected frequencies should be ≥ 5 for valid test

10. Simple Linear Regression
Regression Model & OLS
Goodness of Fit (ANOVA)
Assumptions & Violations
Model:
Yᵢ = b₀ + b₁Xᵢ + εᵢ
  • b₀ = intercept; b₁ = slope coefficient
  • OLS: minimizes Σ(Yᵢ − Ŷᵢ)² (sum of squared residuals)
  • b₁ = Cov(X,Y) / Var(X)
  • b₀ = Ȳ − b₁X̄
  • t-test for b₁: H₀: b₁ = 0; df = n−2
  • Predicted value: Ŷ = b₀ + b₁X
  • R² = SSR/SST = % of variation in Y explained by X (0 to 1)
  • For simple regression: R² = ρ²_XY
  • SST = SSR + SSE (Total = Regression + Error)
  • F = MSR / MSE tests if all slope coefficients = 0
  • SEE = √MSE = standard error of the estimate
  • Lower SEE → better fit
Classical Assumptions:
  • Linear relationship Y~X
  • X values are non-stochastic
  • ε: E(ε) = 0, constant variance (homoskedastic), uncorrelated, normally distributed
Violations (identified by residual plots):
  • Heteroskedasticity: non-constant error variance → biased standard errors
  • Autocorrelation: correlated errors; Durbin-Watson test; common in time series

Big Data Characteristics
Machine Learning Types
Fintech Applications & Risks

  • Velocity: high-speed data generation and processing

13. Understanding Business Cycles
Business Cycle Phases
Economic Indicators
Unemployment Types
  • Expansion: rising GDP, employment, inflation; consumer/business confidence high
  • Peak: maximum economic activity; inflation often highest
  • Contraction/Recession: GDP declining (2+ consecutive quarters); rising unemployment
  • Trough: lowest point; turning point for recovery
  • Recovery: growth from trough; unemployment still high initially
Leading indicators (predict future direction):
  • Stock prices, yield curve spread, building permits, money supply (M2), consumer confidence index
Lagging indicators (confirm trend after the fact):
  • Unemployment rate, CPI, commercial loans outstanding, average duration of unemployment
Coincident indicators (move with economy):
  • GDP, employment level, personal income, industrial production
  • Frictional unemployment: between jobs; always present even at "full employment"; voluntary and temporary
  • Structural unemployment: skills mismatch due to technological change; requires retraining
  • Cyclical unemployment: due to economic downturn; target of fiscal/monetary policy
  • Natural rate of unemployment = frictional + structural (cyclical = 0)
  • Output gap = actual GDP − potential GDP
Economics · Reading 14
14. Fiscal Policy
Objectives & Stance
Fiscal Policy Tools
Multiplier & Limitations
Goals:
  • Full employment, price stability, economic growth
Stance:
  • Expansionary: increase spending or cut taxes → stimulate GDP; may increase deficit
  • Contractionary: cut spending or raise taxes → reduce inflation; may create surplus
  • Structural budget balance: actual balance adjusted for cycle; measures true fiscal stance
  • Discretionary fiscal policy: deliberate changes (stimulus packages, tax cuts)
  • Automatic stabilizers: unemployment insurance, progressive taxation — automatically cushion cycle without new legislation
  • Government spending: transfer payments, public goods, infrastructure
  • Tax policy: income tax, capital gains tax, VAT, corporate tax
  • Fiscal multiplier = 1/(1−MPC) → each dollar of gov't spending increases GDP by more than $1
  • Ricardian equivalence: consumers anticipate future taxes to repay deficit → save more now → spending stimulus offset → fiscal policy ineffective
  • Crowding out: gov't borrowing ↑ interest rates → ↓ private investment → partial offset of stimulus
  • Implementation lags: recognition lag + legislative lag + impact lag → reduces effectiveness

15. Monetary Policy
Central Bank Objectives & Tools
Policy Transmission
Effects & Limitations
Primary objective: price stability (inflation target ~2%)
Also: full employment, financial system stability, GDP growth

Tools:
  • Policy interest rate: main lever (e.g., federal funds rate); most direct tool
  • Reserve requirements: minimum reserves banks must hold
  • Open market operations: buy bonds → inject money; sell bonds → drain money
  • Quantitative easing (QE): large-scale asset purchases when rates near zero
Expansionary monetary policy (lower rates):
  • ↓ Borrowing cost → ↑ consumption and investment
  • ↓ Exchange rate → ↑ net exports
  • ↑ Asset prices → wealth effect → ↑ spending
Contractionary monetary policy (raise rates):
  • ↑ Cost of credit → ↓ spending and investment → ↓ inflation
Neutral rate: neither expansionary nor contractionary
  • Recognition lag: time to identify that action is needed
  • Response lag: time for central bank to act (shorter than fiscal)
  • Impact lag: 12–18 months for full economic effect
  • Liquidity trap: rates near zero; QE may have diminishing returns; cash hoarding
  • Cannot simultaneously target both interest rates AND money supply
  • Inflation targeting: anchor inflation expectations

16. Introduction to Geopolitics
Geopolitical Risk Factors
Tools of Geopolitics
Investment Implications
  • National security: military alliances, territorial disputes
  • Trade & economic policy: sanctions, tariffs, embargoes
  • Political stability: regime type, election risk, social unrest, policy continuity
  • Nationalization risk: government seizure of foreign assets
  • Resource allocation: control of strategic resources (oil, rare earth)
  • Cooperative tools: treaties, alliances, multilateral agreements (WTO, UN)
  • Economic coercion: sanctions, tariffs, foreign aid conditions, debt leverage
  • Military tools: deterrence, use of force, arms sales
  • Countries balance economic interdependence vs national security
  • Globalization increases both cooperation opportunities and vulnerability
  • Geopolitical events → asset price volatility → higher risk premium
  • Supply chain disruptions → commodity price shocks
  • Safe-haven flows: USD, JPY, CHF, gold appreciate in crises
  • Country risk analysis: assess political stability for EM investments
  • Portfolio managers should monitor geopolitical risks as part of risk management

17. International Trade
Comparative Advantage
Trade Barriers
Trade Agreements
  • Absolute advantage: produce more output with same inputs
  • Comparative advantage: lower opportunity cost; true basis for trade; both countries can gain
  • Free trade maximizes global output even if one country has absolute advantage in all goods
  • Heckscher-Ohlin model: export goods using abundant factors (e.g., capital-rich country exports capital-intensive goods)
  • Terms of trade = export price index / import price index
  • Tariff: tax on imports; raises domestic price; government revenue; deadweight loss
  • Import quota: quantity limit on imports; similar price effect but no tariff revenue
  • Export subsidy: government support to domestic producers; distorts trade
  • Voluntary Export Restraint (VER): exporting country limits own exports
  • All barriers: reduce consumer surplus and global efficiency
  • Preferential trading agreement: lower tariffs for select partners
  • Free trade area (FTA): no tariffs among members; each keeps own external tariffs (e.g., NAFTA/USMCA)
  • Customs union: FTA + common external tariffs toward non-members
  • Common market: customs union + free movement of labor and capital
  • Economic union: common market + harmonized economic/monetary policies (e.g., EU)
Economics · Reading 18
18. Capital Flows and the FX Market
Balance of Payments (BOP)
Exchange Rate Regimes
FX Determinants
  • Current account (CA): trade in goods/services + investment income + transfers
  • Capital account (KA): capital transfers, non-produced/non-financial assets
  • Financial account (FA): FDI, portfolio investment, reserve asset changes
  • BOP identity: CA + KA + FA = 0 (always balances)
  • CA deficit → FA surplus (foreign capital inflow)
  • CA surplus → FA deficit (capital outflow/accumulation of foreign reserves)
  • Free float: market-determined; most major currencies (USD, EUR, GBP, JPY)
  • Managed (dirty) float: central bank intervenes to smooth excessive volatility
  • Fixed (pegged) rate: pegged to another currency; requires reserve management
  • Currency board: strict fixed rate; domestic currency fully backed by foreign reserves
  • Crawling peg: small, pre-announced periodic adjustments
  • Interest rate differentials: higher rates attract capital → currency appreciates
  • Inflation: higher inflation → purchasing power erosion → currency depreciates (PPP)
  • Economic growth: strong growth → more investment demand → currency appreciates
  • Trade balance: CA surplus → appreciation
  • Central bank intervention: buy/sell currency to influence rate
  • Speculation and sentiment: short-term driver
Economics · Reading 19
19. Exchange Rate Calculations
Spot Rates & Cross Rates
Forward Exchange Rates
Parity Conditions
  • Spot rate: rate for immediate currency exchange
  • Convention: price currency / base currency (e.g., USD/EUR = USD per 1 EUR)
  • To convert: multiply spot rate by base currency amount
  • Cross rate: exchange rate between two currencies derived from their rates vs a third currency
  • If USD/EUR = 1.10 and USD/GBP = 1.25 → GBP/EUR = 1.10/1.25 = 0.88
  • Contract to exchange currency at future date at a pre-agreed rate
  • Forward premium: forward rate > spot rate (price currency's interest rate is higher)
  • Forward discount: forward rate < spot rate
  • No-arbitrage forward rate:
F/S = (1 + r_price) / (1 + r_base)
  • Forward premium ≈ interest rate differential between the two countries
Purchasing Power Parity (PPP):
  • Absolute PPP: exchange rate = ratio of price levels (S = P_A/P_B)
  • Relative PPP: % change in rate ≈ inflation differential (e_A/B ≈ π_A − π_B)
Interest Rate Parity (IRP):
  • Forward premium reflects interest rate differential between countries
International Fisher Effect:
  • Interest rate differentials reflect expected inflation differentials
  • Real interest rates equal across countries in equilibrium
